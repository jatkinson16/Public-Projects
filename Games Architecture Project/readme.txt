# Games Architecture ACW
Contains the Full source code and release application for my ACW submission for Module 600098 Games Architecture at University of Hull
Made using C# OpenTk and OpenGl

## Folders
**Initial Framework**
Contains all source code for the project as well as all assets

**Run**
Download this to play a release version of the game

## Usage
**In Game**
Controls:
wasd or arrow keys: Move forward or backward, and turn from side to side

Goals:
Collect 3 Keys and reach the exit portal, whilst avoiding the enemy moons.
