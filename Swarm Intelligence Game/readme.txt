# Swarm Intelligence Game (Flocking)
Contains the Full source code and release application for my honours stage project that was to develop a swarm ai in a video game.
Made with Unity/C#

## Folders
**Initial Framework**
Contains all source code for the project as well as all assets

**Run**
Download this to play a release version of the game

## Usage
**In Game**
Controls:
wasd keys: Move your character
Aim your gun with the mouse and left click to shoot!

Goals:
Collect 5 keys and reach the exit

There are currently 3 Levels in the game
